一、安装ES集群

es官网docker安装es集群指南

  1. 在linux下新建目录/opt/elasticserch，下载此链接下 .env 和 docker-compose.yml 文件并上传到新建的目录中
  2. 修改 .env 文件，在 ELASTIC_PASSWORD= 和 KIBANA_PASSWORD= 后面添加设置密码，修改 STACK_VERSION={version} 为 STACK_VERSION=9.0.0
  3. 使用 docker cpmpose 启动
    1. 启动：sudo docker compose up -d
    2. 停止：sudo docker compose down（保留数据）
    3. 删除es：sudo docker compose down -v （不保留数据）
  4. 登录 kibana 后台面板：
    浏览器打开 http://IP:5601 访问后台，用户名为 elastic 密码为刚刚在 .env 文件中的密码
